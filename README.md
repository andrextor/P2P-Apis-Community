Welcome to the **Official Public Feedback Hub** for the P2P-Apis project.

## 🌉 The Communication Bridge

This repository serves as the official point of contact between our community and the core development team. 

> [!IMPORTANT]
> The core source code of **P2P-Apis** is private for security and intellectual property reasons. This hub allows you to contribute to the project's evolution without needing access to the private internals.

---

## 🛠 How to Contribute

We value your feedback and suggestions. You can contribute in the following ways:

### 1. Suggest Features
If you have an idea for a new endpoint, a new payment method, or a performance improvement, please use our [Feature Request Template](https://github.com/andrextor/P2P-Apis-Community/issues/new?template=feature-request.yml).

### 2. Report Bugs
Found something wrong? Help us fix it by providing a detailed report using our [Bug Report Template](https://github.com/andrextor/P2P-Apis-Community/issues/new?template=bug-report.yml).

### 3. Discuss
Join the conversation in the [Discussions](https://github.com/andrextor/P2P-Apis-Community/discussions) tab for general queries and architectural brainstorming.
